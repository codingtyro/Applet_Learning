### Java程序的基本组成

1. 注释
   ~~~ java
   有三种写法
   1.单行注释（//）
   2.多行注释（/* 和 */）注释界定符将一段比较长的注释括起来
   3.文档注释符，可以用来自动生成文档，这种注释以/**开始，以*/结束。
   注意：在Java中，/* */注释不能嵌套，因为这段代码本身也有可能包含一个*/界定符。
   ~~~
2. 数据类型
   ~~~ java
   Java是强类型语言。这就意味着必须要为每一个变量声明一种类型。
   Java共有8种基本类型，其中整型四种，两种浮点型，1种字符char类型和1种用于表示真值的boolean类型。
   在Java中，所有数值类型所占据的字节数与平台无关。
   1.整型用于表示表示没有小数部分的数值，允许是负数
   （1）int <--> 4字节
   （2）long <--> 8字节
   （3）short <--> 2字节
   （4）byte <--> 1字节
   注意：Java没有任何无符合（unsigned）形式的int、long、short或者byte
   2.浮点数用于表示有小数部分的数值。在Java中有两种浮点数。
   （1）float <--> 4字节
   （2）double <--> 8字节
   注意：float类型的数值有一个后缀F或者f（例如3.14f）。没有后缀F/f的浮点数总是默认为double类型，也可以在浮点数值后面添加后缀D/d
   (3)有三个表示溢出或者出错情况的三个特殊的浮点数值：
   <1>正无穷大
   <2>负无穷大
   <3>NaN(不是一个数字)
   3.char类型原本用于表示单个字符，如今，有些Unicode字符可以用一个字符表示，另外一些Unicode字符需要两个char值（这个地方涉及到编码格式的选择）。
   4.boolean（布尔）类型有两个值：false和true，用来判断逻辑条件。整型值与布尔值之间不能进行相互转换。
   ~~~
   3. 变量与常量
   ~~~ java
   1.Java也使用变量来存储值。常量就是值不变的变量。
   2.对于局部变量，如果可以从变量的初始值推断出它的类型，就无须声明类型，这个时候可以使用关键字var
   3.在Java中，并不区分变量的声明和定义。
   4.利用关键字final指示常量。（常量名通常使用全大写）。
   ~~~
   4. 枚举类型
   ~~~ java
   有的时候，变量的取值范围只在一个有限的集合当中，针对这种情况，可以使用自定义枚举类型。枚举类型包括有限个命名的值
   enum Size{SMALL,MEDIUM,LARGE,EXTRA_LARGE};
   现在声明这种类型的变量：
   Size s  = Size.MEDIUM;
   Size类型的变量只能存储这个类型声明中给定的某个枚举值，或者特殊值null，null表示这个变量没有设置任何值。
   ~~~
   5. 算术运算符
   ~~~ java
   加减 乘除运算，整数的求余操作（有时称为取模）
   需要注意，整数被零除将会产生一个异常，而浮点数被零除将会得到无穷大或者NaN结果。
   ~~~
   6. 数值类型之间的转换
   ~~~ java
   1.当使用一个二元符连接两个值的时候（例如n + f，n是整数，f是浮点数），先要将两个操作数转换为同一种类型，然后再计算。
   2.强制转换类型，要注意的是不要在boolean类型与其他数值类型之间进行强制转换，这样可以防止发生一些常见的错误。
   ~~~
   7. 位运算
   ~~~ java
   1.处理整型类型时，可以直接对组成整数的各个位完成操作。这意味着可以使用掩码技术得到整数中的各个位位运算包括|,&,^,~
   2.应用布尔值上，&和|运算符也会得到一个布尔值，这些运算符与&&和||运算符很相似，不过&和|运算符不采用“短路”方式来求值，也就是说，得到结果之前两个操作数都需要计算。
   3.>>>运算符会用0补充高位，这与>>不同，它会采用符号位填充高位，不存在<<<运算符。
   4.移位运算的右操作数需要完成模32的运算（除非左操作数是long类型，在这种情况下需要对右操作数模64）。例如1 << 35的值等同于1 << 3。
   ~~~
   8. 字符串
   ~~~ java
   1.与绝大多数的程序设计一样，java语言允许使用+号连接（拼接字符串）
   2.当将一个字符串与一个非字符串的值进行拼接，后者会转换成字符串（任何一个java对象都可以转换成字符串）。
   3.如果需要把多个字符串放在一起，用一个界定符分隔，可以使用静态join方法：
   String all = String.join("/","s","m","l","xl");
   4.String类没有提供修改字符串中某个字符的方法。
   ~~~
   9. 文件输入与输出
   ~~~ java
   1.想要读取一个文件，需要构造一个Scanner对象
   Scaner in = new Scanner(Path.of("myflie.txt"),StandardCharsets.UTF_8);
   如果文件名中包含反斜杠符号，就要记住在每个反斜杠之间再加一个额外的反斜杠转义
   2.想写入文件，就需要构造一个PrintWriter对象
   PrintWriter out = new PrintWriter("myfile.txt",StandardCharsets.UTF_8);
   如果文件不存在，创建文件。
   ~~~
   10. 大数
   ~~~ java
   1.如果基本的整数和浮点数精度不能满足需求，那么可以使用java.math包中两个很有用的类，BigInteger和BigDecimal,这两个类可以处理包含任意长度的数字序列和数值。BigInteger类实现任意精度的整数运算，BigDecimal实现任意精度的浮点数运算。
   ~~~
